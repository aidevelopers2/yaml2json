# Contributing

Contributions welcome — bug reports, YAML edge cases that break the converter, and small fixes especially.

- **Found a file that converts wrong?** Open an issue with the YAML snippet (redact anything sensitive). These are the most valuable reports.
- **Small fixes** (typos, obvious bugs): just open a PR.
- **Bigger changes**: open an issue first so we can talk about it before you spend time.

Run tests with `npm test` (Node 20+, no other setup).

By contributing you agree your work is released under the MIT license of this repo.
